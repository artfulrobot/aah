The size of a rem is set by the font-size on the html tag. So this box is 10px high not 16px.

1rem - 10px

The problem is that 'rem' is document-wide. Once defined, by font-size on the html tag, it cannot be changed. Changing the font-size on the body tag does change the font size that is inherited, down the chain, but it does not change what 1rem evaluates to.. BS3 includes this html rule.
